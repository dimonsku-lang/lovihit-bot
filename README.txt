ИНСТРУКЦИЯ ПО ЗАПУСКУ БОТА «ЛОВИХИТ»

1. Установите зависимости:
   pip install aiogram gspread oauth2client

2. В config.py вставьте:
   - Ваш Telegram Bot Token от BotFather
   - Ссылку на Google Таблицу (можно оставить пока пустой, если таблицы ещё нет)

3. Запустите бота:
   python main.py

4. Бот будет приветствовать, показывать категории, товары и собирать заявки на рекламу.

Всё просто. Если что — обращайтесь!
